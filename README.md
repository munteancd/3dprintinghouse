# 3D Printing House

Catalog online de printare 3D la comanda — Mosnita Veche / Timisoara.

Site: https://munteancd.github.io/3dprintinghouse/

Modelele din catalog sunt creatii open-source ale comunitatii, folosite sub
licente Creative Commons (CC0 / CC-BY / CC-BY-SA), cu atribuirea autorilor
afisata pe fiecare produs. Comenzile se preiau pe WhatsApp.
